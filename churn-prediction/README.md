## Project Overview
An end-to-end Machine Learning pipeline to predict which customers are likely to cancel their service. This model helps the marketing team target high-risk users with retention offers.

## Key Features
* Automated data preprocessing and scaling.
* Random Forest Classifier with 89% AUC.
* Feature Importance analysis to identify drivers of churn (e.g., Monthly Bill, Support Calls).

## Tech Stack
* Python (Pandas, NumPy)
* Scikit-Learn
* Joblib (Model Serialization)
